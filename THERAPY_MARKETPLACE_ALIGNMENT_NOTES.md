# Therapy + Marketplace Alignment Update

## Therapist and Coach side
- Public discovery is intended to follow the therapy-directory pattern.
- Onboarding copy now frames coaches and therapists as directory providers.
- Dashboards now describe public alignment: verification, service mode, booking flow, and local discovery.
- Booking modal now uses date-and-time scheduling instead of instant session launch language.
- Profile preview no longer redirects straight into a live session after booking.

## Course and Creator side
- Public courses and creators are positioned as a marketplace/storefront.
- Creator dashboard now speaks in product, category, video, and storefront language.
- Courses page and creators page use stronger marketplace search/filter presentation.

## Remaining work outside this patch
- Full visual parity with the uploaded reference videos still needs a full build-and-test pass.
- Some secondary provider pages and dashboard subsections still need deeper UI matching.
- Payment, email, KYC webhook, and session-opening automation still need live backend verification in a full environment.
